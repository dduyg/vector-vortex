Here's a working prototype with Python using synthetic data (GeoPandas + Matplotlib):

import geopandas as gpd import matplotlib.pyplot as plt from shapely.geometry import LineString, Point import pandas as pdSynthetic railway lines data = { 'name': ['Ankara-Sivas', 'Kayseri-Sivas', 'Istanbul-Ankara (Pre-1923)'], 'year_built': [1926, 1930, 1910], 'geometry': [ LineString([(32.85,39.93), (37.02,39.75)]), LineString([(35.48,38.72), (37.02,39.75)]), LineString([(28.98,41.01), (32.85,39.93)]) ], 'ideology_tag': ['Halkçılık', 'Devletçilik', 'Yabancı sermaye'] } railways = gpd.GeoDataFrame(data, crs='EPSG:4326') regions_data = { 'region': ['Ankara', 'Sivas', 'Kayseri', 'Istanbul'], 'literacy_1923': [20, 10, 15, 40], 'literacy_1938': [60, 45, 55, 75], 'geometry': [Point(32.85,39.93), Point(37.02,39.75), Point(35.48,38.72), Point(28.98,41.01)] } regions = gpd.GeoDataFrame(regions_data, crs='EPSG:4326') fig, ax = plt.subplots(figsize=(10,8)) colors = {'Halkçılık': 'green', 'Devletçilik': 'blue', 'Yabancı sermaye': 'red'} for _, row in railways.iterrows(): ax.plot(*row.geometry.xy, color=colors[row.ideology_tag], linewidth=3, label=row.ideology_tag) regions.plot(ax=ax, color='orange', markersize=100) ax.set_title('Republican Railway Expansion and Ideological Purpose') plt.show()

This is just a prototype—you can replace synthetic data with real TCDD, TÜİK, or TBMM data and even build a dashboard using Plotly Dash or Streamlit.
